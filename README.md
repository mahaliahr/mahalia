# mahalia

Portfolio website for Mahalia Henry-Richards

## Development

### Setup
```bash
npm install
```

### Run locally
```bash
npm run dev
```

This generates the image list and starts a local server at `http://localhost:8000`

### Build
```bash
npm run build
```

Generates `assets/images.json` from images in `assets/main-page/`

## Tech
- Vanilla JS with p5.js for interactive elements
- SVG text mask overlay
- Dynamic image loading from JSON
